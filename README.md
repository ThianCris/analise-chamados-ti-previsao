# 📊 Análise e Previsão de Chamados de TI

Este projeto realiza a análise de chamados técnicos ao longo dos anos e utiliza dados históricos para prever a demanda futura.

## 🚀 Objetivo
Analisar o volume de chamados e gerar previsões para auxiliar no planejamento da equipe de TI.

## 🛠️ Tecnologias
- Python
- Pandas
- Matplotlib
- Google Colab

## 📈 Resultados

- Identificação de padrões de crescimento
- Estimativa de chamados para 2025 e 2026
- Apoio na tomada de decisão
